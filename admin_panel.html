API CONFIGURATION
Configure the connection to your bot's API server.
The bot must be running with the updated bot.py that includes the Flask API.
SETUP INSTRUCTIONS
1. Upload the updated bot.py to your server
2. Run: pip install flask flask-cors
3. Start bot: python bot.py
4. The API starts automatically on port 5000
5. Enter your server IP above (e.g. http://1.2.3.4:5000)
6. Enter your API_SECRET from bot.py
7. Click Test Connection — then Save Config
